# Odyssey Demo

![alt text](image.png)

You need [Python 3.13](https://www.python.org/downloads/) to run this demo.

## Install dependencies

```bash
python -m .venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run the server

```bash
fastapi run
```
